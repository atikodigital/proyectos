<a id="stack"></a>
## 7. 🛠️ Stack técnico recomendado para implementar todo

### Modelos IA
- **Claude (Anthropic)** — mejor en español, más obediente, recomendado default
- **GPT-4 / GPT-5.1 (OpenAI)** — más creativo, ecosistema enorme
- **Gemini (Google)** — más barato, mejor con Google Workspace
- **Open-source:** Llama 3, Mistral, DeepSeek (self-hosted)

### Orquestación / Automation
- **n8n** — más flexible, self-hosted gratis, recomendado
- **Make.com** — visual-first, 3000+ integraciones
- **Zapier** — el más conectado (8000+ apps), más caro
- **Activepieces** — alternativa open-source
- **Workato** — enterprise tier

### Cuál elegir según caso
| Necesidad | Mejor opción |
|-----------|--------------|
| Pyme pequeña, flujos simples | Make.com |
| Pyme creciendo, flujos complejos | n8n self-hosted |
| Empresa establecida, sin tech team | n8n Cloud o Workato |
| Quieres control total y precio bajo | n8n self-hosted |
| Necesitas el ecosistema más grande | Zapier |

### WhatsApp / Mensajería
- Twilio (más fácil)
- Meta Business API directa (más barato a escala)
- WhatsApp Cloud API

### CRM
- HubSpot (free tier amplio)
- Notion (CRM ligero)
- Pipedrive
- Salesforce (enterprise)

### Storage / Docs
- Google Workspace (Sheets, Drive, Docs)
- Notion
- Airtable
- Coda
